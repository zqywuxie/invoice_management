# 电子发票汇总程序 - 启动说明

## 环境要求

- Python 3.9 或更高版本
- Windows / macOS / Linux

## 安装依赖

```bash
pip install -r requirements.txt
```

## 启动程序

```bash
python main.py
```

## 功能说明

- **添加发票**: 点击"添加发票"按钮，选择PDF文件导入
- **查看发票**: 在表格中浏览所有发票记录
- **搜索发票**: 在搜索框输入关键词筛选
- **删除发票**: 选中发票后点击"删除"按钮
- **导出Excel**: 点击"导出Excel"按钮生成报表

## 数据存储

发票数据保存在 `data/invoices.db` SQLite数据库中。
